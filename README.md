# Bushe 

Комплексная система аналитики обучения (Learning Analytics), состоящая из двух приложений: десктопного дашборда для администраторов и мобильного приложения для сотрудников. Проект превращает данные об обучении в активный инструмент управления образовательным процессом с помощью машинного обучения и геймификации.

## О проекте

Система разработана для поддержки процесса обучения сотрудников розничного направления (сеть пекарен "Буше"). Вместо замены существующей платформы (iSpring), проект предлагает **дополнение**, которое решает проблемы вовлеченности и забывания материала.

Проект состоит из двух взаимосвязанных приложений, написанных на Python с использованием библиотеки PyQt:

1.  **HR Learning Dashboard (Десктоп):** Инструмент для администраторов и HR-специалистов для мониторинга прогресса, аналитики и получения рекомендаций от ИИ.
2.  **Мобильное приложение (для сотрудников):** Инструмент для закрепления знаний и регулярного повторения материала в игровом формате.

## Психологический подход к целевой аудитории

При разработке функционала мобильного приложения учитывались психологические особенности возрастной группы **19–25 лет** (ранняя взрослость). На основе модели взросления IDEA-R были выявлены ключевые особенности и предложены механики для работы с ними:

*   **Субъективное ощущение свободы выбора** → В приложении реализована возможность "примерять" разные профессиональные роли (работник торгового зала, продавец-кассир, администратор) в игровом формате. Это помогает сотрудникам структурировать представления о карьерном развитии.
*   **Тревожность из-за неопределенности и высокой нагрузки** → Приложение интегрировано с корпоративным чат-ботом, который отправляет напоминания о необходимости прохождения тестов. Это снижает потребность в самостоятельном контроле и уменьшает тревожность, связанную с самоорганизацией.
*   **Лиминальность (переходный период, поиск идентичности)** → Система достижений, уровни и игровые результаты создают безопасное пространство для формирования профессиональной идентичности. Сотрудники могут видеть свой прогресс и ощущать развитие навыков.
*   **Потребность в социальной поддержке и одобрении** → Внедрена система рейтингов и сравнения результатов. За призовые места сотрудники получают премии или сертификаты на покупки в "Буше", что стимулирует вовлеченность и мотивацию.

## Функциональные возможности

### HR Learning Dashboard (Десктоп приложение для администраторов)

*   **Мониторинг прогресса:** Отслеживание уровня усвоения материала по каждому сотруднику и компании в целом.
*   **Визуализация данных:** Построение графиков и рейтингов для наглядного представления статистики.
*   **Прогнозирование забывания (XGBoost):** Модель машинного обучения анализирует более 30 тысяч взаимодействий (интервалы повторений, время ответа, правильность) и предсказывает вероятность забывания учебного элемента.
*   **Интерпретируемость модели (SHAP):** Расчет вклада каждого признака. Анализ показал, что главный фактор усвоения — **скорость ответа в прошлый раз**. Быстрый правильный ответ = глубокое усвоение, медленный ответ = риск забывания.
*   **Формирование рекомендаций:**
    *   Персонализированные списки фактов для повторения ("повторить сегодня").
    *   Выделение "красной зоны" (усвоение < 30%).
    *   Определение самых сложных тем.

### Мобильное приложение (для сотрудников)

*   **Игровой формат обучения:** Короткие интерактивные задания и тесты для регулярного повторения материала.
*   **Система ролей:** Возможность в игровой форме попробовать себя в разных профессиональных ролях (РТЗ, кассир, администратор).
*   **Прогресс и достижения:** Система уровней и наград для формирования ощущения развития профессиональных навыков.
*   **Интеграция с чат-ботом:** Автоматические напоминания о необходимости прохождения тестов для снижения когнитивной нагрузки.
*   **Рейтинги:** Сравнение результатов с коллегами, система поощрений (премии, сертификаты) для стимуляции мотивации.

## Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/DariaVazh/Bushe.git
    cd Bushe
    ```

2.  **Установите зависимости:**
    Рекомендуется использовать виртуальное окружение.
    ```bash
    pip install -r requirements.txt
    ```
    *(Основные зависимости: `PyQt5`, `psycopg2`, `pandas`, `xgboost`, `shap`)*

3.  **Настройка базы данных:**
    *   Убедитесь, что PostgreSQL установлен и запущен.
    *   Создайте базу данных для проекта.
    *   Настройте параметры подключения к БД в конфигурационном файле приложения.

4.  **Запуск приложений:**
    *   Для запуска десктопного дашборда администратора:
        ```bash
        python main.py
        ```
    *   Для запуска мобильного приложения (эмуляция на десктопе или перенос на мобильное устройство):
        ```bash
        python mobile_app.py
        ```

## Ожидаемые результаты внедрения

*   **Замкнутый цикл обучения:** Игровой процесс стимулирует активность → аналитическая система интерпретирует данные → администраторы получают инструменты для управления обучением.
*   **Для сотрудников:** Повышение вовлеченности, снижение забываемости, формирование профессиональной идентичности через геймификацию.
*   **Для HR и администраторов:** Прозрачность процесса, раннее выявление проблемных тем и отстающих, обоснованные рекомендации.
*   **Для компании:** Комплексная адаптивная образовательная среда, повышение качества профессиональных компетенций.
